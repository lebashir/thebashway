# lessons

Distilled build pitfalls for THIS project. Each entry is one `- [tag] rule` line
(tag = an area, or `general`). thebashway injects the relevant ones into each
basha's prompt as "Known pitfalls — do not repeat," and appends a new line when a
gate or a reviewing basha catches a real mistake (dedup is automatic, so the same
pitfall logs once). Start empty; it grows itself. General, cross-project lessons
graduate into the thebashway skill over time.
